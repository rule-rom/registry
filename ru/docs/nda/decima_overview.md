## 🏗️ Что такое Decima8?

**Decima8** — это нейроморфное вычислительное ядро, работающее на принципах детерминированного затухания и активации через пороговые функции.

### Ключевые концепции

| Концепция | Описание |
|-----------|----------|
| **Тайл** | Минимальная вычислительная единица с локальной памятью весов и состоянием |
| **Веса** | Матрица связей 8×8, определяющая трансформацию входного сигнала |
| **Затухание (Decay)** | Детерминированное уменьшение активности тайла во времени |
| **Порог активации** | Диапазон значений, при котором тайл «защёлкивается» в активное состояние |
| **Процесс пропекания (Baking)** | Фиксация интента в субстрате — перевод конфигурации в неизменяемое состояние |

### Архитектурные принципы

```
┌─────────────────────────────────────────────────────────┐
│  Decima8 Resonance Swarm                                │
│                                                         │
│  ┌─────────────┐     ┌──────────────┐     ┌─────────┐  │
│  │   IDE /     │ →   │   Emulator   │ →   │  Swarm  │  │
│  │  Editor     │     │  (SHM IPC)   │     │  Core   │  │
│  └─────────────┘     └──────────────┘     └─────────┘  │
│         ↑                    ↑                  ↑       │
│    Конфигурация      Детерминированный    Физическое  │
│    (Bake Blob)       цикл (40µs target)   исполнение  │
└─────────────────────────────────────────────────────────┘
```

### Паспортные данные

| Компонент | Метрика | Статус |
|-----------|---------|--------|
| **IDE / Emulator** | ~40µs на цикл | Target (i5-3550) |
| **UDP Cascade** | ~20µs latency | Target |
| **Точность** | Level16 (0..15) | Детерминировано |
| **Масштаб** | До 256 тайлов в массиве | Конфигурируемо |
