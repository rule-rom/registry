# 🔒 Доступ ограничен: Tier 4+ (Swarm Founder)

## Автономный эмулятор Decima8

Эта страница содержит спецификацию автономного эмулятора Decima8 с интеграцией через Shared Memory.

### Что внутри

- **Standalone Emulator** — автономная версия эмулятора для развёртывания в коммерческих кластерах
- **Shared Memory Integration** — документация по интеграции через разделяемую память
- **Performance Profiles** — профили производительности для различных конфигураций
- **Deployment Guide** — руководство по развёртыванию в production

### Архитектура эмулятора

```
┌─────────────────────────────────────────────────────────┐
│  Decima8 Emulator (Standalone)                          │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐  │
│  │ SHM Interface│  │ Tile Engine  │  │ BUS16 Model  │  │
│  │ (IPC)        │  │ (8×8 Matrix) │  │ (8 lane)     │  │
│  └──────────────┘  └──────────────┘  └──────────────┘  │
│         ↓                ↓                  ↓           │
│  ┌─────────────────────────────────────────────────┐   │
│  │ EV_FLASH Controller (READ→WRITE→READOUT)       │   │
│  └─────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

### Интеграция через Shared Memory

| Параметр | Значение |
|----------|----------|
| SHM размер | 64 KB (configurable) |
| Выравнивание | 4096 bytes (page-aligned) |
| Протокол | Lock-free ring buffer |
| Задержка | <1 µs на операцию |

### Требования к доступу

| Tier | Доступ |
|------|--------|
| Tier 3 (Industrialist) | ❌ Недоступно |
| Tier 4 (Swarm Founder) | ✅ Эмулятор + SHM интеграция |
| Tier 5 (Swarm Node) | ✅ + исходники эмулятора |
| Tier 6 (High Council) | ✅ полный доступ + модификации |

### Лицензия на развёртывание

**Tier 4+** получают лицензию на развёртывание "Пропечённых Личностей" в коммерческих кластерах:

- ✅ Неограниченное количество инсталляций
- ✅ Право на модификацию конфигурации
- ✅ Право на интеграцию с собственным ПО
- ❌ Запрещена обратная разработка
- ❌ Запрещена передача третьим лицам

### Как получить доступ

1. Станьте спонсором уровня **Tier 4 (Swarm Founder)** или выше
2. После подтверждения оплаты вы получите:
   - Лицензионный ключ
   - Бинарники эмулятора
   - Документацию по интеграции
